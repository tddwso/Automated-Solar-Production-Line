# Automated Solar Production Line
一.整線概況說明

太陽能廠自動化生產輸送線是由PLC系統所控制，目的是將玻璃片一站接著一站往下傳遞，經過各個站點進行製程，最後由人員將太陽能片成品裝箱儲存。
